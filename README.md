RESTful Async Transcription Service using GCP Speech-To-Text API

## Install the dependencies
```bash
make dependencies
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
make dev-client
```

### Start the server
```bash
make dev-server
```

### Build the app for production
```bash
make build
```
